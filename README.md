### 使用說明
本機器人可用以下三種方法查詢你要的咖啡店： 
 <ol>
<li>使用捷運站查詢：請依「查捷運 XXX」格式，輸入查詢。</li>
<li>使用店名查詢：請依「查店名 XXX」格式，輸入查詢。</li>
<li>使用位置查詢：請分享現在位置查詢。</li> 
</ol>
